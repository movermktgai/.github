# Mover Marketing AI GitHub Defaults

This public `.github` repository provides the organization profile and default GitHub community files for repositories in the `movermktgai` organization.

The public presentation is intentionally brand-safe: it supports Mover Marketing AI's agency identity, technical SEO/AEO trust signals, and responsible open collaboration without exposing client details or private operations.

## What this repository controls

- `profile/README.md` — public organization profile shown on `https://github.com/movermktgai`.
- `.github/ISSUE_TEMPLATE/` — default issue templates used when a repository does not define its own templates.
- `SECURITY.md` — default public security-reporting guidance.

## How organization defaults work

GitHub uses files from this repository as organization defaults when an individual repo does not define its own version.

For issue templates:

- Defaults live in `.github/ISSUE_TEMPLATE/` in this repo.
- A repo-specific `.github/ISSUE_TEMPLATE/` folder overrides these defaults.
- Use repo-specific templates when a repository needs tailored fields.

## Public safety rules

Do not add any of the following to this public repository:

- client names, URLs, screenshots, reports, analytics, strategy, or deliverables unless explicitly approved for public use;
- secrets, API keys, deploy hooks, webhook URLs, tokens, credentials, or environment values;
- private monitoring, deployment, indexing, reporting, or automation details;
- internal TODOs, prompt residue, agent notes, or unpublished operational plans.

Public files should stay focused on high-level agency identity, trustworthy SEO/AEO positioning, generic community guidance, and safe contact paths.
