# Mover Marketing AI GitHub Defaults

This public `.github` repository provides default GitHub community files for repositories in the `movermktgai` organization.

## How this works

GitHub uses files from this repository as organization defaults when an individual repo does not define its own version.

For issue templates:

- Defaults live in `.github/ISSUE_TEMPLATE/` in this repo.
- A repo-specific `.github/ISSUE_TEMPLATE/` folder overrides these defaults.
- Use repo-specific templates when a client site or app needs tailored fields.

Do not add secrets, client-private details, unpublished strategy, tokens, webhook URLs, or credentials to this public repository.
